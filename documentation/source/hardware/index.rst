Hardware
########

Hardware documentation for Thermoquad systems.

.. note::
   This documentation is under construction.

Platform
********

Thermoquad targets the Raspberry Pi Pico 2:

* **RP2350A** (Cortex-M33) - Development boards
* **RP2354A** (Cortex-M33F) - Production boards

Board Definitions
*****************

Custom board definitions are located in the ``boards/`` directory.

Components
**********

Helios ICU Hardware
   Temperature sensors, motor drivers, pump control, glow plug relay.

Slate Controller Hardware
   TFT display, touch input, UART interface.
