Slate Controller
################

Slate is the display and control interface firmware.

.. note::
   This documentation is under construction.

Overview
********

Slate provides:

* LVGL-based graphical interface
* Real-time telemetry display from Helios
* User controls for temperature and operation
* Serial communication via Fusain protocol

Hardware
********

* Raspberry Pi Pico 2 (RP2350A)
* TFT display with touch input
* UART connection to Helios ICU
